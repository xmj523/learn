1.黄金分割：0.618/1 = 0.382/0.618 ≈ 0.618
    （上下左右4条线 黄金比例计算:在长边一端点取长度为短边的长度取点，连线，短边从2端点按0.382:0.618计算取点连线）
2.三分线--（上下左右4条线 4条3等分线）
3.黄金螺旋线->最佳兴趣点(4点 2条对角线加4条4角作垂线的交点)
4.斐波那契螺旋线（斐波那契数列）-> 黄金螺旋线
    斐波那契数列（第n(n>=2)项等于前两项之和）：1、1、2、3、5、8、13
5.字号大小：
    选择字号时，通常会选择最大或最小字号做参考，按黄金比1:1.618得到比它大的字体，按1:0.618得到比它小的字体
    使用斐波那契比：1:1/1:2/1:3/2:3/1:5/2:5/3:5...
